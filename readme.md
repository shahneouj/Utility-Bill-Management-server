## Features
- **Firebase Authentication** (Token verification via `firebase-admin`)
- **CRUD Operations** for Bills & Payments
- **User Management** (Auto-create on first login)
- **Secure Routes** (Protected with Firebase ID token)
- **MongoDB Integration** (with Stable API)
- **Serverless Ready** (Vercel, Render, Railway)
- **CORS Enabled**
- **Environment-based Config** (`.env`)

---

## How to Run

1. Clone the repository
2. Install dependencies with `npm install`
3. Set up environment variables (`.env` file):
   - `VITE_FIREBASE_API_KEY` and other Firebase config
   - `MONGODB_URI` for MongoDB connection
4. Start the backend server with `node server.js`
5. Run the frontend with `npm run dev`
6. Open `http://localhost:5173` in your browser
